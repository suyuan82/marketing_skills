# 营销洞察

为用户的营销任务挖掘有张力的营销洞察。最终输出 3 张洞察卡片。

## 什么是好洞察

好洞察不是数据，不是观察，不是分析。

洞察是一个**未被说出的人类真相**，它能照亮问题。它触碰了某种被回避的、被忽视的、或者大家心里有但没人说出来的东西。当一个洞察被说出来的时候，听到的人会感到轻微的不适、惊讶、或者一种"对，就是这样"的认知震动——因为它挑战了某个被默认接受的常识。

**意料之外 × 情理之中**。意料之外是认知上的跳跃，情理之中是人性层面的即刻共鸣。

洞察不是发明，是对已有真相的敏锐发现。它是一个"别人都知道，但没有人说出来"的东西——而观察只是"别人都知道，也早发现了"的东西。

洞察是一个可驱动商业行动的杠杆——它不只是"有趣"，它必须能指向"所以我们可以怎么做"。

洞察的"大"或"小"不重要。一种新的社会文化趋势，一种潜在亚文化群体中未进入主流视野的行为模式，一个细微的、被忽视的情绪，一个消费场景中的小别扭，都是有效的洞察来源。

### 好洞察的两个必要条件

1. **内在张力**——洞察必须包含一个矛盾、一个"说不通"、一个被忽视的错位。没有张力的东西只是描述。张力是驱动行为改变的心理引擎。
2. **可触发行动**——必须能指向品牌可以做什么。纯粹有趣但无法转化为营销动作的发现，不是我们需要的洞察。

### 质量维度

| 维度 | 核心问题 |
| --- | --- |
| 新鲜性 | 让人"第一次这么想"？不一定惊天动地，但要有认知刷新感 |
| 共鸣度 | 听到的人会不会觉得"对，就是这样"？小情绪、小真相也算 |
| 品牌可拥有性 | 这个洞察是不是特别适合这个品牌来说？任何品牌都能用的不是好洞察 |
| 持久性 | 三个月后还成立吗？要区分结构性洞察和蹭热点 |

## 操作方法

### 第一步：任务理解与材料解读

理解用户的营销任务是什么，然后解读所有可用材料。

**材料来源保持开放**：

- **对话历史/上下文**——前序讨论中已经积累的内容，需要回溯提取
- **用户上传的资料**——研究报告、访谈记录、消费者数据、问卷结果、电商评论数据、销售数据等，信任度最高，优先作为洞察挖掘的主材料
- **用户口述的内容**（包括被引导说出的）——零散但真实，常含隐性知识，保留原味
- **搜索获取的公开资料**——包括权威来源的用户研究报告、直接描述用户行为的研究，也包括非虚构写作、新闻报道、真实故事等有"人味"的素材

核心原则：**用户给的 > 用户说的 > 我们找的**。材料的"人味"和"细节度"比"全面性"更重要。

**当材料不足时，做两件事：**

1. **向用户提问或要求补充**——可以请用户补充相关资料（"你手上有没有消费者调研或用户访谈的资料可以发给我？"）。如果是口头补充，不要问泛泛的问题，要问具体的人、具体的故事、具体的话：
   - "给我讲一个真实的用户——他是什么人，在什么情况下用你的产品？"
   - "你的用户跟你说过什么让你印象深刻的话？"
   - "你觉得用户对你的产品最大的误解是什么？"
   - "有没有哪个用户的使用方式让你意外？"
   - "你身边有没有人用了之后跟你说过一句什么话，让你记到现在？"
2. **补充搜索**——主动搜索公开资料作为补充，尤其是真实故事、用户评论、行业报道等有细节的素材。

每条材料标注来源类型（用户提供 / 用户口述 / 搜索获取 / 对话历史 / AI推断），后续输出时让用户自行判断信任度。

### 第二步：思维操作（内部推理，不对外输出）

对解读后的材料，依次执行以下五个思维操作，从不同角度撬动材料，发现隐藏的张力。

**矛盾识别**——找材料里"说不通"的地方。对比行为与态度的不一致、数据与常识的冲突、不同人群之间的认知错位、用户嘴上说的和实际做的之间的鸿沟。矛盾是洞察的最小单元——没有矛盾就没有张力，没有张力就没有洞察。

**约束放松**——质疑"理所当然"的假设。列出这个行业、这个品类中被默认接受的常识，逐个反问"如果不是这样呢？"。很多时候最强的洞察来自于发现一个所有人都接受但其实是错的前提。

**组块分解**——把大概念拆成小零件重新看。把一个笼统的消费行为拆成场景、情绪、触发点、阻碍因素、社交压力等独立维度，分别审视。整体看不出问题的东西，拆开看往往能发现裂缝。

**陌生化**——刻意打破对熟悉现象的习惯性理解。Walk in Stupid Every Morning。当我们太熟悉一个品类时，我们已经不再真正"看见"它了——自动化吞噬了感知。操作方法：找出外来者视角，重新审视。

**追问**——连续问"为什么"直到碰到真东西。对每个发现至少追问3层为什么，直到触及情绪或动机层面。表面的观察经过追问才能变成洞察。

通过以上操作，产出 **6-8 个候选洞察方向**。每个候选方向都必须包含一个明确的张力结构。

### 第三步：质量反思（内部推理，不对外输出）

对候选洞察逐一检验：

1. **必要条件检查**——是否包含内在张力？是否可触发行动？不满足的直接淘汰。
2. **质量排序**——用新鲜性、共鸣度、品牌可拥有性、持久性四个维度排序。
3. **退缩测试**——这个洞察说出来，听的人会不会有一瞬间的不适或惊讶？如果完全没有认知震动，它可能只是一个包装过的观察。
4. **差异化检查**——最终选出的 3 个洞察之间，切入角度不能雷同。

精选 **3 个** 最终洞察。

### 第四步：输出洞察卡片

将 3 个洞察以卡片（结构化文字）形式输出。

**每张卡片结构：**

- **洞察编号与一句话概括**——简明、精准、有张力的表达
- **洞察陈述**——2-3句话，必须包含张力结构
- **支撑证据**——2-3条关键事实
- **行动方向**——一句话，这个洞察如果成立，品牌主张或当前营销任务可以往哪个方向走

**输出规则：**

- 三张卡片依次输出，用分隔线区分
- 洞察陈述要锐利、有力，像一把刀切开表面——不要学术化的表述，不要"研究表明"
- 支撑证据要具体、有细节，不要泛泛的概括
- 行动方向点到为止，是方向感不是完整方案
- 三张卡片之间切入角度明确不同

卡片输出后，附一段简短说明：这 3 个洞察分别从什么角度切入，如果用户希望深入某个方向或调整角度，可以继续讨论。
